# Campus Communication Portal

## Project Description
This mobile application is developed using the **Flutter** framework as part of the **CS310 Mobile Application Development** course at Sabancı University. The platform serves as a centralized hub for the campus community, providing real-time updates and fostering interaction among students and staff. The application integrates **Firebase** for secure user authentication and scalable data storage.

---

## Project Features

### Core Features (Minimum Viable Product)
* **Secure Authentication**: User registration and login strictly restricted to `@sabanciuniv.edu` email addresses via Firebase.
* **Real-Time Campus Feed**: A chronological stream where users can share immediate updates such as shuttle delays or library occupancy.
* **Club Announcement Module**: A dedicated and verified section for student organizations to publish event details and news.

### Optional Features (Nice-to-Have)
* **Integrated Marketplace**: A campus-specific second-hand market allowing students to list and browse items for sale.
* **Interaction System**: Features such as "likes" to validate and highlight the most relevant campus posts.

---

## Technical Stack
* **Frontend**: Flutter
* **Backend**: Firebase (Firestore & Authentication)
* **Version Control**: Git and GitHub
